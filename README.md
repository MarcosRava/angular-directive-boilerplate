# es6-angular-directive-boilerplate

A simple boilerplate to build directives with es6 using webpack, karma and jasmine

### Usage
```
cd angular-directive-boilerplate
npm install
```

### Run tests
```
npm test
```

### Creating es5 bundle
```
npm run build
```
